## 1. Product Overview
- 一个 3D 零件展示网站，使用 Three.js 将图片零件转换为可交互式 3D 立体模型并支持旋转、缩放等操作
- 面向工程设计、产品展示等场景，提供沉浸式的 3D 可视化体验

## 2. Core Features

### 2.1 Feature Module
1. **3D 展示页**: 3D 零件模型展示、交互控制、环境设置
2. **控制面板**: 旋转控制、缩放控制、视角重置

### 2.3 Page Details
| Page Name | Module Name | Feature description |
|-----------|-------------|---------------------|
| 3D 展示页 | 3D 场景 | 零件模型渲染、光照效果、相机控制 |
| 3D 展示页 | 交互控制 | 鼠标拖拽旋转、滚轮缩放、平移 |
| 3D 展示页 | 控制面板 | 自动旋转开关、视角重置按钮、光照调节 |

## 3. Core Process
用户打开页面 → 加载 3D 零件模型 → 通过鼠标/触摸设备交互 → 控制模型旋转、缩放、平移 → 使用控制面板调整视图

```mermaid
graph TD
  A[用户打开页面] --> B[加载 3D 场景]
  B --> C[渲染零件模型]
  C --> D[用户交互操作]
  D --> E{操作模型旋转/缩放]
  E --> F[使用控制面板]
  F --> G[调整视图]
```

## 4. User Interface Design

### 4.1 Design Style
- 主色调：深蓝色 (#0f172a)、金属银 (#e2e8f0)
- 辅助色：蓝色 (#3b82f6)、深灰 (#334155)
- 按钮风格：圆角矩形、悬浮效果
- 字体：Inter，现代无衬线字体
- 布局风格：全屏 3D 场景 + 固定位置控制面板
- 图标风格：简洁现代线性图标

### 4.2 Page Design Overview
| Page Name | Module Name | UI Elements |
|-----------|-------------|-------------|
| 3D 展示页 | 3D 场景 | 深色渐变背景、金属质感零件、环境光、定向光 |
| 3D 展示页 | 控制面板 | 半透明玻璃质感、固定在右下角、圆角设计 |

### 4.3 Responsiveness
- 桌面优先，支持触摸设备
- 移动端适配：响应式控制面板
- 响应式设计，支持多种屏幕尺寸

### 4.4 3D Scene Guidance
- 环境/HDRI：深色工业风格
- 光照设置：多光源系统（环境光、定向光、点光源）
- 相机设置：透视相机，初始角度，自动旋转
- 交互：拖拽旋转、滚轮缩放、右键平移
- 动画：模型自动旋转、平滑过渡
